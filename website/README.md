# agentOrc website

Official documentation and marketing site for **agentOrc** — the local-first semantic memory SDK for AI agents.

**Live:** [https://AgentOrc.lucareo.com](https://AgentOrc.lucareo.com)

## SEO surfaces this site owns

| Path | Purpose |
| --- | --- |
| `/` | Brand + product home (canonical) |
| `/docs/*` | Indexed documentation |
| `/benchmarks` | Public benchmarks + FAQ schema |
| `/sitemap.xml` | Full crawl map |
| `/robots.txt` | Crawl policy |
| `/llms.txt` | AI / LLM citation summary |
| `/manifest.webmanifest` | PWA / brand metadata |
| `/opengraph-image` | Social share card |
